## code_015
再帰での階乗の計算<br>
例えば、num=4の時<br>
num * factorial(num - 1)<br>
4 * factorial(3)<br>
3 * factorial(2)<br>
2 * factorial(1)<br>
retuurn 1<br>
<br>
つまり、4 * 3 * 2 * 1になる<br>

動的計算法（dynamic programming）<br>
漸化式を利用して、次々に途中までの解を記録していき<br>
最終的に解を求める方法
